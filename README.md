<h1 align="center"> SISTEMA DE GERENCIAMENTO DE PRODUTOS </h1>

O projeto desenvolvido para a disciplina de PROGRAMAÇÃO ORIENTADA A OBJETOS II, o sistema tem o objetivo de gerenciar os produtos de uma empresa de tecnologia de pequeno porte como forma de facilitar o cotidiano da empresa.

##  Funcionalidades do projeto
O sistema permitirá que faça:
1. ***Cadastro de novos produtos***: ID, preço, nome e quantidade
2. ***Atualização***: Atualizar informações de produtos que já existe
3. ***Exclusão***: Excluir os produtos do estoque
4. ***Consulta e Visualização dos produtos***
5. ***Persistência de Dados***: Arquivo CSV (armazenamento permanente)

## Tecnologias:

- JAVA
- _JAVAFX_
- POO
